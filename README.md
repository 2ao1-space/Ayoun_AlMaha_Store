A full-featured optics store web application for selling **medical glasses**, **sunglasses**, managing **doctors discounts**, **repairs system**, and **virtual try-on** experience.

Built with a modern stack focused on **speed, scalability, and simplicity**.

---

## Tech Stack

### Frontend

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **GSAP** (Animations)
- **Zustand** (State Management)
- **React Hook Form + Zod**
- **Lucide Icons**

### Backend (BaaS)

- **Supabase**
  - Authentication
  - PostgreSQL Database
  - Storage
  - Row Level Security (RLS)

---

## Project Goals

- Fast MVP delivery
- Clean & scalable architecture
- Simple admin management
- User-friendly experience
- Modular system (easy future upgrades)

## Features

### User Side

- Landing Page

- User Authentication
- Cart & Favorites
- Orders & Checkout
- Doctors Discounts (Prescription upload & approval)
- Repairs Requests & Tracking
- User Profile

### Admin Side

- Products CRUD
